Descrição

O ficheiro suru-plus-folders é um executável bash que permite que você modifique a cor dos diretórios nos temas de ícone Suru++ 25 e Suru++ Ubuntu, baseado no clone de ícones de diretórios do tema de ícones Papirus.

Neste momento, suru-plus-folders não possui uma interface gráfica do utilizador, mas é uma aplicação de linha de comando totalmente funcional. Antes de visualizar os exemplos de utilização, por favor, instale primeiramente:

Instalação

Distribuição agnóstica

Utilize o script para instalar a versão mais recente directamente do repósitorio (independentemente da sua distribuição):

A instalar

wget -qO- https://git.io/fhQdI | sh

Para instalar suru-plus-folders nos sistemas BSD, utilizando comando seguinte:

wget -qO- https://git.io/fhQdI | env PREFIX=/usr/local sh

A desinstalar

wget -qO- https://git.io/fhQdI | env uninstall=true sh

Alguns exemplos de utilização:

  • Mostrar a cor atual e as cores disponíveis para Suru++ 25 ou Suru++ Ubuntu: 
    # Suru++ 25
suru-plus-folders -l --theme Suru++
# Suru++ Ubuntu
suru-plus-folders -l --theme Suru++-Ubuntu
  • Modificar a cor de diretórios para brown (marrom) para Suru++ 25 ou Suru++ Ubuntu: 
    # Suru++ 25
suru-plus-folders -C brown --theme Suru++
# Suru++ Ubuntu
suru-plus-folders -l --theme Suru++-Ubuntu
  • Revertir para a cor padrão de diretórios para os ícones de places do Suru++ 25 ou Suru++ Ubuntu: 
    # Suru++ 25
suru-plus-folders -D --theme Suru++
# Suru++ Ubuntu
suru-plus-folders -l --theme Suru++-Ubuntu
  • Restaurar a última cor utilizada a partir dum ficheiro de configuração: 
    suru-plus-folders -Ru

Advertência importante!

Isto é extremamente útil para restaurar a cor após cada atualização de tema de ícones (instaladores oficias de Suru++ e alguns pacotes terceirizados fazem isto automaticamente)..

OBSERVAÇÃO: Para modificar a cor de um diretório individual, você pode utilizar Folder Color ou Dolphin Folder Color.
